# Marwa Mahmoud — Portfolio

Personal site for **Marwa Mahmoud Elsayed**, frontend engineer focused on Angular, React, Next.js, and bilingual EN/AR (RTL) product work.

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion
- lucide-react

## Requirements

- Node.js 20+
- [pnpm](https://pnpm.io/) 9 or 10 (this repo pins `pnpm@10.15.1` in `packageManager`)

## Run locally

```bash
pnpm install
pnpm dev
```

Open the URL Vite prints (usually `http://localhost:5173`).

## Build

```bash
pnpm build
pnpm preview
```
